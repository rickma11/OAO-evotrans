# OAO-FC26 UT模式翻译插件

## 1. 插件概述
**OAO-FC26 UT模式翻译插件**是一款专为 FC26 UT模式玩家设计的 Tampermonkey 脚本，用于将 Futbin 和 Fut.gg 网站的英文内容翻译成中文，帮助中文用户更便捷地浏览和使用这些网站。

### 1.1 支持的网站
- **Fut.gg**：包括所有子页面
- **Futbin.com**：包括所有子页面

## 2. 核心功能
### 2.1 翻译功能
2.1.1 基本信息翻译

2.1.2 六维属性翻译

2.1.3 角色翻译

2.1.4 化学风格翻译

2.1.5 球员特技翻译

2.1.6 FUTGG/FUTBIN网站专用翻译

2.1.7 进化名称翻译


### 2.2 侧边栏功能
2.2.1 更新翻译按钮
- 手动刷新进化翻译数据
- 支持多数据源自动切换

2.2.2 FUTGG翻译开关
- 控制 Fut.gg 网页的翻译功能
- 默认打开
- 点击后实时关闭/打开翻译
- 状态保持（刷新页面后仍保持设置）

2.2.3 FUTbin翻译开关
- 控制 Futbin 网页的翻译功能
- 默认打开
- 点击后实时关闭/打开翻译
- 状态保持（刷新页面后仍保持设置）

### 2.3 技术特点
2.3.1 多数据源支持
- 主数据源：Gitee（国内访问更快）
- 备用数据源：GitHub
- CDN数据源：jsdelivr

2.3.2 缓存机制
- 进化翻译数据缓存24小时
- 减少重复请求，提高性能

2.3.3 动态内容翻译
- 使用 MutationObserver 监听DOM变化
- 自动翻译动态生成的内容

2.3.4 响应式设计
- 侧边栏可拖拽调整位置
- 适应不同屏幕尺寸

2.3.5 错误处理
- 多数据源自动切换
- 详细的错误日志和用户提示


## 3. 安装与使用

### 3.1 安装步骤
1. 安装 Tampermonkey 浏览器扩展
2. 在 实用工具-导入 OAO-evotrans.user.js 脚本

### 3.2 使用方法
1. 访问 Fut.gg 或 Futbin.com 网站
2. 侧边栏会自动显示
3. 使用侧边栏按钮控制翻译功能
4. 点击"更新翻译"按钮获取最新进化翻译数据
